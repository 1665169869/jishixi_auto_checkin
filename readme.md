# 技实习自动签到打卡

`API`是广州市工贸技师学院的技实习接口，其他学校请自行修改`baseUrl`

## 吐槽

这是我弄过最麻烦的一次，登录接口需要四次请求不同的api才能拿到完整cookie，屎山代码啊，吐了

## 使用方法

1. 下载项目到本地
2. 修改 `index.js` 文件，填写自己的学号和密码
3. 在命令行中运行 `node index.js` 即可

```js
// index.js
const USERS = [
  {
    user_account: '学号',
    psw: '密码'
  }
]
```
